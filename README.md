## Driving School Appointment Learning System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/751f3fad3e0e446eab2bd674cd6b4095/ghb20250919) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can be used)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Students, Instructors, Administrators.

**Student Functions:**
1. Login
2. Registration
3. Personal Center (Change Password, Edit Personal Information, Change Avatar)
4. Instructor Appointment
5. Study Material Favorites
6. Exam Records
7. Error Question Bank
8. Homepage
9. Exam Information
10. Instructor Information
11. Study Materials
12. Message Board
13. Announcement Information

**Instructor Functions:**
1. Login
2. Registration
3. Personal Center (Change Password, Change Avatar, Edit Personal Information)
4. Announcement Information Overview
5. Instructor Appointment View
6. Message Board
7. Study Materials
8. Test Paper Management
9. Test Question Management

**Administrator Functions:**
1. Login
2. Homepage
3. Personal Center
4. Administrator Management
5. User Management
6. Instructor Management
7. Announcement Information Management
8. Instructor Appointment Management
9. Message Board Management
10. Study Material Management
11. Test Paper Management
12. Test Question Management
13. Exam Management
14. Basic Data Management (Announcement Types, Subjects, Study Material Types, Paper Assembly Methods)
15. Banner Management
